--------------------------------------- HOW TO PLAY 2048 --------------------------------------------------
1. Open the terminal of the program.
2. Type "python 2048.py".
3. Once you type that a menu will appear, it will show you the options that it offers that looks like this.
   +-----------------------------+
   |          HOW TO PLAY        |
   |  Use ↑ ↓ ← → to move        |
   |  Combine same tiles         |
   |  Press R to restart         |
   |  Press Q to quit            |
   |  Press ENTER to start       |
   +-----------------------------+
4. Follow the guide and press enter to start the game.
5. When you start the game it will start witha block that has the value of 2 the your slide it to whatever direction you want using the arrows on your keyboard.
6. Just play and slide non stop.
7. There is no win in this and you will only be able to finish the game when you lose.
8. Once the game is finished you can choose if you want to restart by pressing "r" or you can quit by pressing "q" on your keyboard.
